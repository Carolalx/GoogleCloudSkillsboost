# App Engine: Qwik Start - Python - GSP067  

1 - No menu de navegação clique em `APIs & Services` > `Library`   
2 - Digite `App Engine Admin API` na caixa de pesquisa   
3 - Clique em `App Engine Admin API card`  
4 - Clique em `manage` ou `enable` para habilitar o API   
5 - Clique `>_` (ao lado do icone do Gemini) para abrir o Cloud Shell   
6 - Copie os comando informados em Task 2. e cole no terminal (um de cada vez)   
7 - Pule para a Task 5. e copie o código:   
```
gcloud app deploy

```

8 - Informe o número conforme a região disponibilizada em seu LAB   
9 - Digite `y` para aceitar a instalação   
10 - Verifique seu progresso
