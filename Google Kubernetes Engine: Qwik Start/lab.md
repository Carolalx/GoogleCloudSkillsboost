# 🌐 Google Kubernets Engine: Quick Start 🚀

1 - Clique em `Start lab`   
2 - Abra o console em janela anonima > insira usuáirio e senha > Clique em `Agree and Continue`   
3 - Clique para abrir o Cloud Shell `>_` (ao lado do ícone do Gemini)   
4 - Copie o código abaixo e cole no terminal


```
curl -LO https://raw.githubusercontent.com/Carolalx/GoogleCloudSkillsboost/refs/heads/main/Google%20Kubernetes%20Engine%3A%20Qwik%20Start/carolalx.sh
sudo chmod +x carolalx.sh
./carolalx.sh
```
5 - Digite `Y` quando for solicitado e pressione `Enter`   
6 - Verifique seu Progresso! (pode levar alguns segundos)
